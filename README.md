# Supermarket Sales Analysis

## Overview
This project analyzes supermarket sales data using Google Sheets.

## Features
- Data Cleaning
- Rating Category Classification
- KPI Dashboard
- Pivot Tables
- Charts
- Business Insights

## Key Findings
- Food and Beverages generated the highest sales revenue.
- Health and Beauty generated the lowest sales revenue.
- Naypyitaw contributed the highest overall sales revenue.
- Ewallet was the most preferred payment method.
- Average customer rating was 6.97.
